Purpose

The goal of the SCI0 Specification Project is to document all resource formats used in the Sierra Creative Interpreter. The homebrew-gaming community has unlocked much of the older AGI engine, and hopefully this document will increase interest in doing the same for SCI.

The Specification is meant as an aid in SCI research and tool design. It does not explain how the interpreter works or how to develop games for the interpreter.



Contributors

Carl Muckenhoupt - As far as I know, one of the first to work with SCI. Much of what I know comes from his SCI Decoder.

Unknown - Discovered the vocabulary resource format.

Dark Fiber - Information about multiple resource listings.

Dark Minister - Information about resource loading / logic resources

How can you help? Well, you could help directly by discovering more about SCI resources and how they get used. Alternatively, you could write some newer, better SCI resource management programs or help test those already in existance.
